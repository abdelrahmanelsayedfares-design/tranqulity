# 🌿 Tranquility App

A wellness and motivational mobile application built with Flutter that helps users improve their mental well-being through inspirational quotes, AI-powered conversations, and a clean and user-friendly experience.

---

## ✨ Features

* 🔐 User Authentication (Login, Register, OTP Verification)
* 🔑 Forgot & Reset Password
* 💬 AI Assistant Chat
* 📖 Inspirational Quotes
* 👤 User Profile Management
* 📝 Suggestions & Feedback System
* 🎨 Responsive and Modern UI
* 💾 Local Data Storage using Shared Preferences
* 📱 Responsive Design for Different Screen Sizes

## 🚀 Run the Project

### Clone the Repository

```bash
git clone https://github.com/abdelrahmanelsayedfares-design/Tranquility.git
```

### Install Dependencies

```bash
flutter pub get
```

### Run the App

```bash
flutter run
```
